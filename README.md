## O que esse repositório é
- A tese e os princípios
- Métricas agregadas de autonomia operacional (sem nomes de cliente, sem código proprietário)

## Por quê

Em janeiro de 2026, decidi parar de ler papers sobre AI agents e tentar **operar uma empresa real só com IA**. Hoje, esse experimento tem 2 clientes pagantes, métricas públicas e nenhum funcionário humano além de mim.

Vou compartilhar:
- A razão de autonomia (autonomy_ratio) diária
- Padrão agregado de falhas
- Decisões de arquitetura de alto nível
- Aprendizados que viraram regra (ex: "auto-fix com revert é melhor que approve-each-step")

Não vou compartilhar:
- Quem são meus clientes
- Código fonte
- Detalhes de orquestração
